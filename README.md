# Emotion_Analyses_project

用PMI方法计算统计10000条外卖评论中50个倾向值最大和最小的单词。

>1.将所有评论用工具分词  
>2.选定正向负向情感种子词各5个  
>3.利用公式计算每个单词的语义倾向值：t = ∑ PMI( word , pword ) - ∑ PMI( word , nword )  
>4.其中，PMI( word1 , word2 ) = log2( ( P( word1 ∩ word2 ) / ( P( word1 ) * P( word2 ) ) ) )  
>5.倾向值最大的50个和最小的50个单词即为所求  

社会计算课程的大作业之一。
